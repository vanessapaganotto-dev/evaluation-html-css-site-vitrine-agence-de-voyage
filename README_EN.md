## Description

Static website built with HTML5 and Bootstrap 5.  
It features a homepage with a header, a responsive navigation menu, an image carousel of the cities of the month, a top 5 destinations section with images and descriptions, and a footer.

---

## Technologies Used

- HTML5  
- Bootstrap 5 (via CDN)  
- Bootstrap Icons (via CDN)  
- Bootstrap JavaScript (via CDN) for the carousel

---

## Installation

Clone this repository:  
```bash
git clone <repository-url>

Open the `index.html` file in a modern web browser.
```

---

## Features

- Responsive navigation menu with toggle on small screens  
- Bootstrap carousel with images and captions  
- Responsive grid layout presenting the top 5 destinations  
- "Read more" buttons for each destination (non-functional)  
- Simple footer with copyright notice
